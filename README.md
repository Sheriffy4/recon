Полностью навайбкоженый проект, вдохновленный легендарным Zapret от Bol-van (где-то в коде даже есть упоминание о нем). Изначально хотелось сделать автоматический поиск стратегий для zapret.
Пока что только под win. Пока что только разблокировка сайтов в браузере (возможности разблокировки discord даже не пробовал добавлять).
Позже добавлю более подробное описание. Создавалось и тестировалось только мной, на python 3.12.9
Есть несколько режимов работы. Режим поиска стратегии (cli.py) и режим обхода (simple_service.py).
Заблокированные домены, которые надо разблокировать, записываются в файл sites.txt.
Еще есть куча неиспользуемого и устаревшего кода. Может быть со временем что-то будет чиститься.
В общем для начала рекомендуется:
1. заполнить нужными доменами файл sites.txt (в данной конфигурации)
2. запустить поиск стратегий обхода  - python cli.py auto -d sites.txt --mode deep и ждать когда все домены протестируются
3. запускаем службу обхода simple_service.py
4. открываем браузер и проверяем работу сайтов
